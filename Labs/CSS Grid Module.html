My demonstration of CSS Grid

This example has 2 columns

The red border is the outer div with the .wrapper class. This is the main container in which the grid is formed.

The green borders are the two (2) nested divs on which the columns are generated based on the grid-template-columns: 70% 30%; property values.

Column 1.
This size is 70%. This column is inside of a div element that is using the .wrapper class. The .wrapper class is the container for the grid. The grid is design using two (2) nested div.
