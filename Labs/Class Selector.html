CSS Selectors - Class

There are three (3) different ways you can select HTML elements for styling.

  • Class - In this method you can assign different classes to our elements.
  • Each element can have multiple classes,and each class can also be applied to multiple elements.
  • You must assign the class className to the element using “=“.
  • In the stylesheet you select the class using .className format.
  • Matches elements based on the contents of their class attribute.
